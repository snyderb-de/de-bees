# Home-page direction explorations

Throwaway visual comps generated 2026-06-15 to explore alternate art directions
for the home page, kept for later reference. **Not used by the app** — each `.html`
is a self-contained standalone mockup (system fonts only, no build step); open it
directly in a browser, or view the matching `.png` render. `_contact-sheet-*.png`
are the two sets stitched together.

Two sets of five. Each set includes one "official state-list" variant.

## Set A — cozy / cute
| File | Direction | Feel |
|---|---|---|
| `a1-honey-kitchen` | Honey Kitchen | Cream, rounded, handwritten headline, friendly cards. Warmest. |
| `a2-storybook-meadow` | Storybook Meadow | Illustrated meadow — sun, hills, flowers. Whimsical. |
| `a3-comb-cottage` | Comb & Cottage | Honeycomb hex tiles, terracotta + honey, typewriter. |
| `a4-friendly-state-registry` | Friendly State Registry ★ | Soft sage seal with a cute bee — government-but-warm. *(official feel)* |
| `a5-picnic-jar-label` | Picnic Jar Label | A literal honey-jar label — gingham, oval label, wax seal. |

## Set B — appeals to beekeepers
| File | Direction | Feel |
|---|---|---|
| `b1-field-notebook` | Field Notebook | Graph paper + hive-inspection log, "flowing now" ticks. Utilitarian. |
| `b2-apiary-coop` | Apiary Co-op | Forest green, stacked Langstroth boxes, "by keepers, for keepers." |
| `b3-old-almanac` | Old Almanac | Two-tone aged paper, woodcut bee, nectar calendar front and center. |
| `b4-state-apiary-registry` | State Apiary Registry ★ | Navy + gold state seal, Dept. of Agriculture letterhead. *(official feel)* |
| `b5-modern-apiary-tool` | Modern Apiary Tool | Clean, filter chips, Delaware map with county pins. |

These explore the cozy/cute lane that `PRODUCT.md` currently lists as an
anti-reference — kept as reference for a possible direction change, not as an
endorsement of one.
